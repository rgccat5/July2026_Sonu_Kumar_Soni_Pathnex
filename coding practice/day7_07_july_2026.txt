# Day 07 - Autmating Cloud Infrastructure with Terraform & Ansible
     * Ansible - Setup Nginx with custom Config
    -name: Setup ngings with coustem config
     host:all
     become: yes
     task:
       -name: Install nginx
          yum:
            name: nginx
            state: present
          -name: Copy custom nginx.conf
           copy:
            src: nginx.conf
            dest: /etc nginx service
          -name: Start nginx service
            Service:
               name: nginx
               state: started 
               enabled: yes
  *Terraform - Provision EC2 with IAM Role
