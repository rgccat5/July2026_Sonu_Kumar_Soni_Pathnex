2: .terraform.lock.hcl : It stores the exact provider version. Every developers uses the same prvider version, prevents unexpected upgrades. Example : 

            provider "registry.terraform.io/hasicrop/aws" { 
                version = "5.92.0" 
             }
3: backend.tf : It sores terraform define state, possible backends: Local, AWS S3, Azure Storage, GCS, Terrafom Cloud
        example :

                 backend "s3" {
                 bucker = "pathnex-devops-terraform"
                 key    = "ansible/terraform.tfstate"
                 regin  = "ap-south-1"

                 }
                }
4: Providers.tf : It specifies which cloud provider terrafrom wil use like AWS, Azure, Google Cloud
                  Example :
                    provider "aws" {
                    tegion = var.region 
                    }
5: variable.tf : Declares all input variables
 
              Example:
                  variable 'region" {
                   type = string
                  }
                  variable "instance_type" {
                   type = string
}
                  varuabke "key_name" {
                   type = string

6: terraform.tfvars : Stroe the actual values for the variables.

7 : data.tf : it contents data sources.
